¡Bienvenido!

Calculadora virtual

Para ejecutar desde la consola (default/por defecto):

Windows: Abre la consola con F12 → pestaña Console.
Mac: Abre la consola presionando ⌘ + Option + J (Command + Option + J).

1. Escribe ' calcular (num1, num2, operacion); ' en la consola.
2. Sustituye 'num1' y 'num2' por 2 números con los que quieras operar (deben ser números enteros -sin decimales-)
3. Sustituye 'operacion' por un operador válido y entre comillas:
"+" para suma
"-" para resta
"*" para multiplicación
"/" para división.

Ejemplo: calcular (1, 2, "+");
En consola debe aparecer el mensaje "El resultado es: 3"

¿Y qué pasaría si num1 o num2 no fueran números (o no fueran enteros)? ¿O si el operador no es válido? ¡Inténtalo!


Para ejecutar como usuario:

Para ello, debes enlazar el script 'prompts-y-alerts.js' en este HTML (en lugar del que ya está enlazado: 'consola.js').
