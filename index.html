¡Bienvenido a la calculadora virtual!

Para ejecutar desde la consola (default/por defecto):

  • Windows: Abre la consola con F12 → pestaña Console
  • Mac: Abre la consola presionando ⌘ + Option + J (Command + Option + J)
  1. Escribe ' calcular (operacion, num1, num2); ' en la consola.
  2. Sustituye 'operacion' por un operador válido y entre comillas:
    "sumar" para suma
    "restar" para resta
    "multiplicar" para multiplicación
    "dividir" para división
  3. Sustituye 'num1' y 'num2' por 2 números con los que quieras operar (deben ser números enteros -sin decimales-)

Ejemplo: calcular ("+", 1, 2,);
En consola debe aparecer el mensaje "El resultado es: 3"



Para ejecutar como usuario:

Para ello, debes enlazar el script 'prompts-y-alerts.js' en este HTML (en lugar del que ya está enlazado: 'consola.js').

  1. Aparecerán "prompts" o ventanas para capturar los datos con los que quieres operar (Operación, Número 1 y Número 2)
  2. Sigue las instrucciones de cada prompt:
    1. Escribe el operador que deseas utilizar o salir para detener la ejecución de la calculadora
    2. Escribe el primer número para ejecutar la operación.
    3. Escribe el segundo número para ejecutar la operación.
    4. ¡Ejecuta cuantas veces quieras!
