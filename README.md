# Aplicación de Consola en JavaScript

Proyecto del **Bootcamp de Desarrollo de Aplicaciones Front-End**.
Módulo 4: _Fundamentos de Programación en JavaScript_.

La aplicación permite realizar operaciones matemáticas básicas, trabajar con condicionales, bucles, funciones, arreglos y objetos, todo ejecutado en el **navegador y/o en la consola del mismo**.

Este proyecto cuenta con 2 scripts:

- **consola.js**: para ejecutar el código en la consola del navegador
- **prompts-y-alerts.js**: para ejecturalo como un usuario (que no necesariamente sabe que existe siquiera la consola)

Las instrucciones de uso se encuentran en el HTML para que el uso sea intuitivo.

---

## Funcionalidades principales

- Operaciones matemáticas: suma, resta, multiplicación y división.
- Validaciones de entrada (ejemplo: evitar división por cero).
- Uso de condicionales (`if`, `switch`) para evaluar situaciones.
- Implementación de bucles (`for`, `while`) para recorrer arreglos.
- Funciones reutilizables para modularizar el código.
- Creación y manipulación de objetos y arreglos de objetos.

---

## Tecnologías utilizadas

- **JavaScript (ES6+)**
- **Visual Studio Code** como entorno de desarrollo
- **Navegador** para ejecutar las funciones de la calculadora con prompts y alerts
- **Consola del navegador** para ejecución en otro entorno

---

## Navegador **(prompts-y-alerts.js)** vs. su Consola del navegador **(consola.js)**

En la consolda del navegador podremos encontrar diferentes ejercicios en los que se evidencian los conocimientos básicos del uso de arrays y objetos, funciones y validaciones de enradas de usuario.

En el navegador sólo podremos interactuar con la calculadora a nivel usuario a través de los prompts y alerts.
